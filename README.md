# 👟 Sneakerhead RAG App

This is a Retrieval-Augmented Generation (RAG) project that answers sneaker-related questions using AI.

## 🚀 Features
- Ask questions about sneakers
- AI answers based on dataset
- FAISS vector search
- HuggingFace LLM integration

## 🧠 Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- HuggingFace

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
